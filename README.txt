## Step-by-Step Launch Guide

### 1. Prepare Your OpenAI API Key
1. Go to https://platform.openai.com/account/api-keys
2. Log in or create an OpenAI account
3. Click **+ Create new secret key**
4. Copy your new key and keep it somewhere safe
5. In the `index.html` file, paste your API key where indicated (backend version optional for security).

---

### 2. Deploy with Vercel (No Code Needed)
1. Go to https://vercel.com
2. Sign in with GitHub
3. Click **New Project** and import your uploaded repository
4. Leave settings default
5. Click **Deploy**
6. You’ll get a live link: `https://yourname.vercel.app`

---

### 3. Connect Your Domain (Loopia)
1. Go to Vercel → Project → Settings → Domains
2. Add: `andreassen.ai` and `preben.ai`
3. Copy DNS instructions from Vercel
4. Log in to Loopia DNS editor
5. Update A or CNAME records as instructed
6. Done! You’re live on your custom domain.
